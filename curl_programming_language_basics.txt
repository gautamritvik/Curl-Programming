"Curl" Programming Language Basics

Name: Curl
Purpose: For fun!
CLI command: curlang [YOUR-FILE].curl
Install: pip install curl-programming-lang

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SYNTAX RULES
  - Every statement ends with \
  - Blocks open with - and close with --\
  - Strings use double quotes: "like this"
  - Variable names need no quotes

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PRINT
  pcType{"Hello, world!"}\
  pcType{"Hello " + var{name}}\
  pcType{var{x} + var{y}}\

INPUT
  pcAsk{"What is your name? >>"}\
  — the answer is later accessed as: input{ans}

VARIABLES
  var{name, "Ritvik"}\             assign a string
  var{score, 100}\                 assign a number
  var{greeting, "Hi " + var{name}}\
  pcType{var{name}}\               print a variable

LISTS
  list{1; 2; 3}\

FUNCTIONS
  createFunc{greet}-
      pcType{"Hello!"}\
  --\
  func{greet}\                     call the function

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

IF / ELIF / ELSE

  1. Basic if:
     if{var{x} == 10, then}-
         pcType{"ten"}\
     --\

  2. If / else:
     if{var{x} == 10, then}-
         pcType{"ten"}\
     else:
         pcType{"not ten"}\
     --\

  3. If / elif:
     if{var{x} > 10, then}-
         pcType{"big"}\
     elif{var{x} == 10, then}-
         pcType{"exact"}\
     --\--\

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

LOOPS

  while loop (repeats while condition is true):
    while{var{x} < 10}-
        pcType{var{x}}\
        var{x, var{x} + 1}\
    --\

  for loop (iterates a variable from start up to, not including, end):
    for{i, 0, 5}-
        pcType{var{i}}\
    --\

OPERATORS
  ==   equals to             !=   not equals to
  <    less than             >    greater than
  <=   less than or equal    >=   greater than or equal
  +    add / concatenate     -    subtract
  *    multiply              /    divide
  \    end of line           -    open block
  --\  close block           {}   argument list
  ""   string data           ;    list separator
  ,    parameter separator

OTHER LANGUAGE BLOCKS
  Supported: "Python", "JavaScript", "Node.js"
  (Java, C, C++ are parsed but not runnable at runtime)

  otherCoding{"Python",

      print("hello from Python")

  }\

  otherCoding{"JavaScript",

      console.log("hello from JS")

  }\

AI MODULE

  pcAI is built-in — no import needed.

  Configure via environment variables:
    CURL_AI_KEY (or OPENAI_API_KEY)   — your API key
    CURL_AI_BASE_URL                  — base URL (default: OpenAI)
                                        use http://localhost:11434/v1 for Ollama
    CURL_AI_MODEL                     — model name (default: gpt-4o-mini)

  Works with: OpenAI, OpenRouter, Ollama, any OpenAI-compatible API.

  Methods:

    pcAI.context{"You are a helpful assistant."}\
      — set a system prompt / bot persona (resets history)

    var{reply, pcAI.ask{"What is 2 + 2?"}}\
    pcType{var{reply}}\
      — ask a question; conversation history is kept across calls

    pcAI.chat{""}\
      — start an interactive chat loop in the terminal (type exit to quit)

    pcAI.reset{""}\
      — clear conversation history (keeps context)

    var{s, pcAI.summarize{"some long text here"}}\
    var{a, pcAI.analyze{"some data"}}\
    var{mood, pcAI.sentiment{"I love this!"}}\   returns: positive / negative / neutral
    var{t, pcAI.translate{"Bonjour to English"}}\

  Full example:
    pcAI.context{"You are a friendly tutor."}\
    var{answer, pcAI.ask{"Explain variables in one sentence."}}\
    pcType{var{answer}}\

  Chat example:
    pcAI.chat{""}\

"Curl" Programming Language Basics

Name: Curl
Purpose: For fun!
Syntax:

	Say something: pcType{[text surrounded by double quotes]}\
	Input: pcAsk{[text surrounded by double quotes]>>}\ (the input answer will later be referred as "input{ans}")
	Storage variable: var{[name of variable, quotes not needed], [variable items, works just like lists]}\ (the variable will later referred as "var{[name of variable]}"
	List: list{[item1]; [item2]; [item3]}\
	Function:
		
			  createFunc{[name of function, quotes not needed]}-     (the function will later be referred as "func{[name of function]}")
(4 spaces or 1 tab) ---->     [code]
			  --\

	Loop:
		1. if{[condition (e.g. var{name} == "Ritvik Gautam")], then}-
		       [code]
		   --\

		2. if{[condition (e.g. var{name} == "Ritvik Gautam")], then}-
		       [code]
		   else:
		       [code]
		   --\

		3. if{[condition (e.g. var{name} == "Ritvik Gautam")], then}-
		       [code]
		   elif{[condition (e.g. var{name} != "Ritvik Gautam")], then}-
		       [code]
		   --\--\

	AI: pcAI{[AI mode surrounded in double quotes (".on" or ".off")], 
	    [AI's directions surrounded in double quotes], [AI modes (modes are: profanityControl, searchPowers, AImager. Toggles are either (in double quotes) ".on" or ".off")]}\
	Blocks of code from other supported coding languages: otherCoding{[Coding language name in double quotes (supported: "Python", "JavaScript", "Java", "C", "C++", "Node.js")],

[code]

}\

	Import a package: import{[package name], [package reference nickname]}\

Symbols:
	== is "equals to"
	= is "set something to"
	!= is "not equals to"
	; is "separation of data, like a comma"
	, is "separation of parameters"
	\ is "end of code line"
	- is "start of code loop"
	-- is "end of code loop"
	{} is "stuff"
	"" is "text data"
